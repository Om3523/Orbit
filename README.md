# 💸 Orbit Finance (Offline Edition)

A beautifully designed, lightning-fast personal finance tracker. Orbit helps you keep a close eye on your income, expenses, and total balance without needing to create an account or connect to a bank.

## ✨ Key Features

* **Starting Balance Configuration:** Set your current bank or wallet amount before you start tracking.
* **Income & Expense Tracking:** Easily categorize your transactions to see where your money is going.
* **Automatic Calculations:** Instantly calculates your total balance, total income added, and total expenses.
* **Transaction History:** View a chronological list of your recent spending and earnings.
* **Data Persistence:** Uses Browser `localStorage` to securely save your financial data locally on your device.
* **Modern UI:** Built with a sleek, dark-mode-first aesthetic using Tailwind CSS.

## 🛠️ Tech Stack

* **Frontend:** HTML5, Vanilla JavaScript
* **Styling:** Tailwind CSS (via CDN)
* **Storage:** Browser Local Storage

## 🚀 How to Run

1. Download or clone this repository.
2. Locate the `index.html` file.
3. Double-click the file to open it in any modern web browser.
4. Click "+ Set Starting Amount" and begin logging your transactions!
